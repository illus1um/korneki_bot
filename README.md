# Korneki Bot

Telegram-бот для бизнеса Караганды — помощь по языковому законодательству, готовые переводы и консультации.

## Возможности

- Двуязычный интерфейс (казахский / русский)
- Законодательство о языках, рекламе, защите прав потребителей, КоАП
- Call-center — бесплатный перевод через WhatsApp
- Готовые переводы для 6 сфер бизнеса (торговля, автосервис, стройматериалы, товары для дома, косметология, меню)
- Контакты специалистов акимата города Караганды

## Технологии

- Python 3.10+
- [aiogram 3.x](https://docs.aiogram.dev/) — асинхронный Telegram Bot API
- python-dotenv — переменные окружения

## Быстрый запуск

```bash
# Клонировать репозиторий
git clone https://github.com/YOUR_USERNAME/korneki_bot.git
cd korneki_bot

# Создать виртуальное окружение
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate     # Windows

# Установить зависимости
pip install -r requirements.txt

# Настроить токен
cp .env.example .env
# Отредактировать .env — указать BOT_TOKEN от @BotFather

# Запустить
python bot.py
```

## Аналитика и отчёты

Бот пишет события в локальную SQLite-базу `korneki.db` (создаётся автоматически рядом с `bot.py`).

Таблицы:
- `users` — `user_id`, `first_seen`, `last_seen`, `lang`, `username`, `first_name`
- `events` — `ts`, `user_id`, `event_type`, `lang`, `section_key`

Типы событий: `start`, `menu`, `help`, `lang_set`, `nav`, `category`, `law_section`, `translation_section`.

### Команда `/stats`
Доступна только Telegram-аккаунтам из переменной `ADMIN_IDS` (через запятую). Показывает: всего пользователей, активных за 24ч / 7д / 30д, языковой сплит, топ-10 разделов за 30 дней.

### CSV-отчёт за период

```bash
python report.py --from 2026-01-01 --to 2026-04-01
```

Создаст `reports/2026-01-01_2026-04-01/` со следующими файлами:
- `summary.csv` — агрегаты (всего пользователей, активных, по языкам)
- `events_by_day.csv` — события и уникальные пользователи по дням
- `new_users_by_day.csv` — новые пользователи по дням
- `top_sections.csv` — топ-100 просмотренных разделов за период

В отчёт попадают только агрегаты — сырых `user_id` нет, что соответствует требованиям по ПДн.

## Деплой

Бот работает в режиме long polling — достаточно запустить `python bot.py` на любом сервере с Python 3.10+.

Для продакшена рекомендуется использовать systemd, Docker или PaaS (Railway, Render).
